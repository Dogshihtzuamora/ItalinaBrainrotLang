ItalinaBrainrotLang Documentation

ItalinaBrainrotLang is an esoteric programming language inspired by "Italian Brainrot" memes, featuring an absurd and humorous syntax for a fun and chaotic programming experience.

Function Definition

To define a function, use the italianaAi keyword followed by the function name. Inside the function, write the body of the function.

italianaAi saudacao()
    talalero tralala -> "Olá, Mundo!"
brain;

Variable Declaration

To create a variable, use the Frulli Frulla -> name -> value; syntax.

Frulli Frulla -> name -> "brainrot";
Frulli Frulla -> age -> 5;

If/Else Conditional Structure

To use the conditional structure, use the Trulimero Trulicina for the if and Frigo Camello for the else. The brain; should be used to terminate the block of code for the condition.

Trulimero Trulicina age > 20:
    talalero tralala -> "Adult"
Frigo Camello;
    talalero tralala -> "Minor"
brain;

Function Call

To call a function, simply use its name followed by parentheses name().

saudacao()
