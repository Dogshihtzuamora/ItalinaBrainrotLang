ItalinaBrainrotLang Documentation

ItalinaBrainrotLang is an esoteric programming language inspired by "Italian Brainrot" memes, featuring an absurd and humorous syntax for a fun and chaotic programming experience.

Function Definition

To define a function, use the italianaAi keyword followed by the function name. Inside the function, write the body of the function.

italianaAi saudacao()
    talalero tralala -> "Olá, Mundo!"
brain;

Variable Declaration

To create a variable, use the Frulli Frulla -> name -> value; syntax.

Frulli Frulla -> name -> "brainrot";
Frulli Frulla -> age -> 5;

If/Else Conditional Structure

To use the conditional structure, use the Trulimero Trulicina for the if and Frigo Camello for the else. The brain; should be used to terminate the block of code for the condition.

Trulimero Trulicina age > 20:
    talalero tralala -> "Adult"
Frigo Camello;
    talalero tralala -> "Minor"
brain;

Function Call

To call a function, simply use its name followed by parentheses name().

saudacao()

While Loop

The Tung Tung Tung Sahur -> keyword creates a loop that continues to execute as long as the given condition is true.

Tung Tung Tung Sahur -> age < 10
    talalero tralala -> "You're young!"
brain;

For Loop

To create a for loop, use the Bombombini Gusini keyword followed by a condition in parentheses Bombombini Gusini (10) and a block of code to execute.

Bombombini Gusini (5)
    talalero tralala -> "Repetition"
brain;

Break Statement

To exit a loop or conditional block, use the Brr Brr Patapim statement.

Tung Tung Tung Sahur -> age < 5
    talalero tralala -> "Breaking loop"
    Brr Brr Patapim;
brain;

File Inclusion

To include and execute external files, use the Serpentini Toiletini -> keyword followed by the file name.

Serpentini Toiletini -> "external_file.italina";

Throw Error

To throw an error, use the Bombardiro Crocodilo -> keyword followed by the error message.

Bombardiro Crocodilo -> "An error occurred!";

Return Value

To return a value from a function or block, use the Lirili Larila -> keyword followed by the value.

Lirili Larila -> "Returned value";
